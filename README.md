# ETH-CA - Acente Üretim Takip Sistemi

Sigorta acenteleri için üretim takip ve hedef yönetim paneli.

## Özellikler

- **Ana Panel**: Toplam prim, poliçe adetleri, branş bazlı gerçekleşme oranları
- **Üretim Ekle**: Şube ve branş bazlı poliçe üretimi kaydetme
- **Şube Yönetimi**: Merkez ve alt şubeleri yönetme
- **Hedef & Basamak**: Şirketin belirlediği basamaklara göre hedef ayarlama
- **Raporlar**: Şube seçerek PDF rapor oluşturma ve indirme
- **Kalıcı Veri**: Tüm veriler localStorage'da saklanır, sayfa kapansa bile kaybolmaz

## Branşlar

Trafik, Kasko, İşyeri, Konut, Sağlık, DASK, Diğer

## Kurulum

```bash
npm install
npm run dev
```
